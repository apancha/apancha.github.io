Weather Application REST

Java Spring Boot MySQL Linux Server REST


(Output to user's phone.)

For my final project as an undergrad student I created a weather application that makes REST calls to provide the end user with weather information. The details for this are as follows:

Developed a back-end REST API using the Java Spring library, that makes calls to DarkSky and Twillo APIs to check the precipitation probability and send a text to the users letting them know whether or not it will rain on that particular day.

The REST API can be controlled with GET,POST,UPDATE, and DELETE functions to manipulate user info that is stored in a database.
